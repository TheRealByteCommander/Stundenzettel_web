# Agent Prompts

Dieses Verzeichnis enthält die Markdown-Dateien mit den System-Prompts für alle Agenten.

## Übersicht

- **`chat_agent.md`**: Prompt für den Chat Agent (Dialog mit Benutzer)
- **`document_agent.md`**: Prompt für den Dokumenten Agent (PDF-Analyse)
- **`accounting_agent.md`**: Prompt für den Buchhaltungs Agent (Zuordnung & Berechnung)
- **`orchestrator_agent.md`**: Richtlinien für den Agent Orchestrator

## Bearbeitung

**Wichtig**: Diese Dateien können jederzeit bearbeitet werden, um das Verhalten der Agenten anzupassen. Änderungen werden beim nächsten Neustart des Backend-Servers oder beim Neuerstellen der Agent-Instanzen wirksam.

## Format

- Markdown-Format
- UTF-8 Kodierung
- Klare Strukturierung mit Überschriften
- Detaillierte Anweisungen für jeden Agent

## Beispiel-Anpassung

Um den Chat Agent freundlicher zu machen, bearbeiten Sie `chat_agent.md` und fügen Sie beispielsweise hinzu:

```markdown
## Kommunikationsrichtlinien:

- Verwende immer höfliche, professionelle Sprache
- Sei besonders geduldig und verständnisvoll
- ...
```

Die Änderung wird beim nächsten Neustart aktiv.
